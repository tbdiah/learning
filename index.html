Tuberculosis Monitoring, Evaluation and Surveillance

Using a Performance-based Monitoring and Evaluation Framework to Strengthen Tuberculosis Programming

This online learning experience is designed for dedicated professionals working to end tuberculosis (TB) in their countries. It provides essential information and guidance on how to effectively monitor and evaluate the performance of TB programs using a performance-based framework. This course requires a basic understanding of TB monitoring and evaluation (M&E) and surveillance systems and explores how a performance-based M&E framework (PBMEF) with standardized, high-quality indicators can improve the generation of quality data and support the analysis, interpretation, and use of that data to inform programming, identify gaps, and strengthen the overall system.
